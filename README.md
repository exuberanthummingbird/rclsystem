## Building and launch.

> напоминаю что лучше не запускать готовые системд вирусы 

### Linux
1. установи gnu toolchain (gcc вирус), qemu x86, xorriso
2. запусти ./build.sh
3. вуаля бесполезная ос готова

### Windows (W.I.P)
рсл систем не будет под винду потому что это тупая ос
